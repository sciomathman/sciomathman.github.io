Reaction of Numbers

CW Reaction: When two numbers of same sings are put to reaction then we have a clockwise reaction that results into a number whose value is the sum of maximum of two numbers and the ratio of minimum number to the maximum number. The sign of the resultant will be same as that of the maximum number

Symbol for CW Reation is ↻

Ex 1: 2 ↻ 5 => 5 + (2/5) = 5.4

Ex 2: -2 ↻ -5 => 5 + (2/5) = -5.4

CCW Reaction: When two numbers of opposite sings are put to reaction then we have a counter clockwise reaction that results into a number whose value is the difference of maximum of two numbers and the ratio of minimum number to the maximum number. The sign of the resultant will be same as that of the maximum number

Symbol for CCW Reaction is ↺

Ex 1: -2 ↺ 5 => 5 - (2/5) = 4.6

Ex 2: 2 ↺ -5 => 5 - (2/5) = -4.6

The BUT Operator

The BUT operator accepts two or more inputs and produce an output

by following the boolean expression Q = A BUT B => NOT(A) AND B

 

Symbol Truth Table

 

2-input BUT Gate
 A B Q
0 0 0
0 1 1
1 0 0
1 1 0
Boolean Expression Q = A.B  NOT of A AND with B results Q

It also makes sense that only when the inputs are contrast with one another BUT

logic works to select the dominant case. Similar to the Exclusive OR logic

we can call BUT logic as Exclusive AND or simply XAND

 

The NBUT Operator

The NBUT operator accepts two or more inputs and produce an output

by  following the boolean expression

Q = A NBUT B => NOT(NOT(A) AND B) => A OR NOT(B)

 

Symbol Truth Table

 

2-input NBUT Gate
 A B Q
0 0 1
0 1 0
1 0 1
1 1 1
Boolean Expression Q = A + B  A OR with NOT of B results Q

NBUT is just the opposite of BUT logic built with OR logic. Here when the

inputs are contrast with one another NBUT logic selects the inferior case.
